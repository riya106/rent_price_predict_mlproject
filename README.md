#  House Rent Prediction

A Machine Learning project that predicts house rent prices based on user inputs like city, area, BHK, bathrooms, and furnishing status.

##  Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn
- Flask
- HTML/CSS

![Homepage](rent1.png)




##  Features
- Rent prediction using regression model
- Cleaned and preprocessed dataset
- Simple web interface using Flask

##  How to Run
1. Clone the repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000/
